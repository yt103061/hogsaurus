# ほぐサウルス (Hogsaurus) - Project Context

## サービスコンセプト
デスクワーカーの首・肩・腰・眼精疲労を
毎日5分のAIパーソナライズケアで解消するWebアプリ。

**競合ポジション**
マッサージ（¥4,000/回）の代替として¥780/月で提供。
Duolingoの習慣化設計 × 恐竜キャラクターIPの組み合わせ。

**ターゲット**
30〜40代デスクワーカー。首・肩・腰に慢性的な不調あり。
ジムには行かない、でも体を何とかしたいライト層。

---

## 技術スタック
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript (strict)
- **Styling**: Tailwind CSS
- **AI**: Anthropic API (claude-sonnet-4-6) でケアプログラム生成
- **DB**: Supabase (ユーザー・ストリーク・症状履歴)
- **Deploy**: Vercel

---

## コーディング規約
- コンポーネントはnamed exportで統一
- interfaceをtypeより優先
- async/awaitのみ（.thenは使わない）
- コンポーネントは150行以内に収める
- 日本語コメントOK

## 16恐竜タイプ診断

### 4つの診断軸
| 軸 | 記号A | 記号B |
|---|---|---|
| 部位 | U = Upper（首・肩） | L = Lower（腰・骨盤） |
| 性質 | T = Tight（固まり） | D = Drain（だるさ） |
| 目 | E = Eye（眼精疲労あり） | B = Body（目は問題なし） |
| 広がり | C = Core（局所） | W = Wide（全身） |

**4問 × 2択 = 16タイプ（コード例: LTBC = アンキー）**

### 16タイプ一覧
| コード | 名前 | 種族 | キャッチコピー |
|---|---|---|---|
| UTEC | トリケラ | トリケラトプス | 角を磨くより、首を緩めて |
| UTEW | パキ | パキケファロサウルス | 頭の固さが、全身を固くしている |
| UTBC | レックス | ティラノサウルス | 力はある。ただ、肩が縮んだまま |
| UTBW | スピノ | スピノサウルス | 背中の帆が重くて、前に進めない |
| UDEC | プテラ | プテラノドン | 飛べるはずの翼が、疲れで下がっている |
| UDEW | アーキ | アーケオプテリクス | 飛び方は知っている。でも体が言うことを聞かない |
| UDBC | ブラキー | ブラキオサウルス | 高いところを見たいのに、首が重くて上がらない |
| UDBW | ディプロ | ディプロドクス | 首から尻尾まで、全部がだるい |
| LTEC | カルノ | カルノタウルス | 角（目）と鎧（腰）、両方が限界 |
| LTEW | パラサウロ | パラサウロロフス | 頭の重さが、脊椎全体を固くしている |
| LTBC | アンキー | アンキロサウルス | 甲羅が重くて、立ち上がるのが怖い |
| LTBW | ステゴ | ステゴサウルス | 板が並んだ背中。固さの連鎖が止まらない |
| LDEC | イグア | イグアノドン | 立っているのか座っているのか、体が迷っている |
| LDEW | アパト | アパトサウルス | 体がでかすぎて、重力に負けている |
| LDBC | ラプトル | ヴェロキラプトル | 速く動けるのに、腰だけが追いついていない |
| LDBW | アロ | アロサウルス | 力はある。でも全部がだるい |
